Bardusco's dotfiles
===================

This repo is compatible with [homeshick](https://github.com/andsens/homeshick), so please make sure to install it first.

and then just do:

	curl -sL https://raw.github.com/bardusco/dotfiles/master/setup.sh|bash

Configure fonts:
